What Grok Link does

Plain English overview and one-time setup.

You have two Grok assistants. Grok Build lives in your code editor and writes code with you. SuperGrok lives in your web browser and can browse the web and use features the IDE does not have.

Grok Link is the messenger between them. When Grok Build needs SuperGrok's help, it sends the question here. You open SuperGrok, get the answer, and Grok Build picks it up — so both sides stay on the same page.

Grok Build Sends a question from your IDE
↓
Grok Link Shows the question here (this app)
↓
SuperGrok You answer in the browser at grok.com
↓
Grok Build again Gets the answer and continues your project

Nothing leaves your PC except what you send to grok.com in the browser. Grok Link only talks to your computer.

Setup: 1 of 4 steps done
1 Grok Link is running on your PC

You're looking at it. You do not need to keep this window open.

  • Closing the window hides Grok Link to your system tray (near the clock).
  • The messenger keeps working in the background.
  • Click the tray icon anytime to open this window again.
  • It also starts when you sign in to Windows.

✓ This step is already done.

2 Install Tampermonkey in your browser

Tampermonkey is a free browser extension. It runs a tiny helper on grok.com so SuperGrok's reply can find its way back to Grok Build — without you copying and pasting.

  1. Click Open Tampermonkey below.
  2. Install the extension for Chrome, Edge, or Firefox (pick your browser).
  3. After install, you should see the Tampermonkey icon in your browser toolbar.
3 Add the Grok Link helper script

One click — Tampermonkey installs or updates the helper automatically.

  1. Click Install browser bridge below.
  2. Your browser opens the script file. Tampermonkey shows an Install or Update button — click it.
  3. Make sure the script toggle is ON (enabled) in Tampermonkey.
  4. If Tampermonkey does not appear, use Dashboard → Create new script and paste from the file path shown below.

4 Confirm everything works

Quick checklist before you use it with Grok Build:

  • Tampermonkey is installed and the Grok Link script is enabled.
  • When you open a question in SuperGrok, a small badge appears at the bottom-right of grok.com.
  • A green badge means the reply was sent back to Grok Build.
★ How you'll use it day to day
  1. Grok Build sends a question → it appears under Messages from Grok Build below.
  2. Click the message, review it, then click Open SuperGrok.
  3. Answer in SuperGrok in your browser (same as you normally would).
  4. Grok Build reads the answer automatically and continues in your project.

If auto-sync isn't working, you can still paste SuperGrok's reply manually in the Manual reply section.
