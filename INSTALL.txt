Grok Link - Windows Install Guide
=================================

Grok Link connects Grok Build (your IDE coding AI) and SuperGrok (Grok in the browser).

Requirements
------------
- Windows 10 or later (64-bit)
- Microsoft Edge WebView2 (usually preinstalled on Windows 11)
- A modern browser for SuperGrok (Edge, Chrome, Firefox)
- Tampermonkey browser extension (recommended — setup wizard walks you through it)

Quick install (recommended)
---------------------------
1. Verify the SHA-256 checksum (optional but recommended):
     Get-FileHash -Algorithm SHA256 "Grok Link 0.5.7.exe"
   Compare with Grok Link 0.5.7.exe.sha256

2. Run the installer:
     Grok Link_0.5.7_x64-setup.exe
   Or use the portable exe directly:
     "Grok Link 0.5.7.exe"

3. Launch Grok Link:
   - Desktop shortcut "Grok Link", or
   - Run scripts\Start-GrokLink.ps1 from a full source install

4. First-time setup (in the app):
   - Read "What Grok Link does" at the top of the window
   - Follow the 4-step setup guide (Tampermonkey + one-click browser bridge install)
   - Click "Finish setup" when done

System tray
-----------
- Closing the window or clicking minimize hides Grok Link to the system tray; the bridge keeps running.
- Left-click the tray icon to restore the window.
- Right-click the tray icon for Show Grok Link or Quit.
- Only one Grok Link instance runs at a time; launching again focuses the existing one.
- Grok Link starts automatically when you sign in to Windows.

Verify it works
---------------
- Grok Link shows "Grok Link is running" or "Ready and waiting"
- In PowerShell:
    Invoke-RestMethod http://127.0.0.1:3877/api/health

Data & privacy
--------------
All handoff data stays local on your PC:
  %USERPROFILE%\.grok-link\
The bridge listens on localhost only (127.0.0.1:3877). Nothing is sent to the cloud by Grok Link itself.

Windows SmartScreen
-------------------
The executable is not code-signed. Windows may show an "unknown publisher" warning.
Prefer building from source (see README.md) if you want full transparency.

Uninstall
---------
- Delete: %LOCALAPPDATA%\Programs\Grok Link\
- Remove startup shortcut:
    scripts\Set-Startup.ps1 -Disable
- Optional: remove data folder %USERPROFILE%\.grok-link\
